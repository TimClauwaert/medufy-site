Medufy
======

*Landing page for Medufy*

http://medufy.com/

## Usage

Set up your environment.

    $ script/bootstrap

Compile your ``scss``.

    $ sass --watch assets/sass:assets/css --style compressed --scss

Run jekyll.

    $ jekyll serve --watch
